CSS Colors

The color CSS property sets the foreground color value of an element's text and text decorations, and sets the currentcolor value.

hsl()

the hsl() functional notation expresses a color in the sRGB color space according to its hue, saturation, and lightness components. An optional alpha component represents the color's transparency

Defining complementary colors with hsl() can be done by adding or substracting 180 degres from the hue value, as they are positioned on the same diameter of the color wheel. For example, if the hue angle of a color is 10deg, its complementary has 190deg as its hue angle.

Description of Values

hue
A number, and angle or the keyword non representing the color's hue angle
saturation
A percentange or the keyword none. This value represents the color's saturation. For example, 100% is completely saturated, while 0% is completely unsaturated (gray)
lightness
A percentage or the keyword none. This value represents the color's lightness. For example, 100% is white, 0% is black, and 50% is "normal"
alpha
An alpha value representing the channel value of the color, where the number 0 correnponds to fully transparent, and 1 corresponds to fully opaque. If the alpha channel value is not specified, it defaults to 1 or 100%.

rgb()

The rgb() functional notation expresses a color in the sRGB color space according to its red, green, and blue components. An optional alpha component represents the color's transparency.

Description of Values

red, blue, green
Each value can be represented as a number between 0 and 255, a percentange between 0% and 100%, or the keyword none. These values represent the red, green, and blue channels, respectively.
alpha
An alpha representing the alpha channel value of the color, where the number 0 corresponds to fully transparent, and 1 to fully opaque. If an alpha channel is not specified, it defaults to 100%

Examples

                    
                        /* Keyword values */
                        color: currentcolor;

                        /*  values */
                        color: red;
                        color: orange;
                        color: tan;
                        color: rebeccapurple;

                        /*  values */
                        color: #090;
                        color: #009900;
                        color: #090a;
                        color: #009900aa;

                        /*  values and legacy  values*/
                        color: rgb(34, 12, 64);
                        color: rgb(34, 12, 64, 0.6);
                        color: rgba(34, 12, 64, 0.6);
                        color: rgb(34 12 64 / 0.6);
                        color: rgba(34 12 64 / 0.6);
                        color: rgb(34.6 12 64 / 60%);
                        color: rgba(34.6 12 64 / 60%);

                        /*  values and legacy  values */
                        color: hsl(30, 100%, 50%);
                        color: hsl(30, 100%, 50%, 0.6);
                        color: hsla(30, 100%, 50%, 0.6);
                        color: hsl(30 100% 50% / 0.6);
                        color: hsla(30 100% 50% / 0.6);
                        color: hsl(30.2 100% 50% / 60%);
                        color: hsla(30.2 100% 50% / 60%);

                        /*  values */
                        color: hwb(90 10% 10%);
                        color: hwb(90 10% 10% / 0.5);
                        color: hwb(90deg 10% 10%);
                        color: hwb(1.5708rad 60% 0%);
                        color: hwb(0.25turn 0% 40% / 50%);

                        /* Global values */
                        color: inherit;
                        color: initial;
                        color: revert;
                        color: revert-layer;
                        color: unset;
