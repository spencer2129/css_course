CSS Font Properties

font-family
Specifies a prioritized list of one or more font family names and/or generic family names for the selected element
font-family: Arial, Helvetica, sans-serif;
font-size
Sets the size of the font
font-size: 12px;
font-size: 0.5em;
font-style
Sets whether a font sohould be styled with a normal, italic, or oblieque face from its font-family
font-style: oblique;
font-style: inherit;
font-weight
Sets the weight (or boldness) of the font. The weights available depend on the font-family that is currently set
font-weight: 600;
font-weight: lighter;
line-height
Sets the height of a line box in horizontal writing modes. In vertical writing modes, it sets the width of a line box.
It is most commonly used to set the height of a line.
line-height: 3em;
line-height: 34%

CSS Text Properties

text-align
Sets the horizontal alignment of the inline-level content inside a block element or table-cell box
text-align: center;
text-indent
Sets the length of empty space (indentation) that is put before the lines of text in a block
text-indent: 15%;
text-indent: 2em;
text-transform
Specifies how to capitalize an element's text. It can be used to make text appear in all-uppercase or all-lowercase, or with each word capitalized
text-transform: capitalize
text-decoration
Sets the appearance of decorative lines on text.
text-decoration: underline;
text-decoration: line-through
text-shadow
Adds shadow to text. It accepts a list of shadows, and each shadow is described by some combination of X and Y offsets from the element, blur radius, and color
text-shadow: 1px 1px 2px black;
text-emphasis
Applies emphasis marks to text
text-emphasis: filled red;
white-space
Sets how white space inside an element is handled
white-space: pre;
white-space: pre-line
