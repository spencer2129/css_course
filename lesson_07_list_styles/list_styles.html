Constituent properties

The list-style CSS property allows you to set all the following list style properties at once

list-style-image
Sets an image to be used as the list item marker
list-style-image: url("/shared-assets/images/examples/rocket.svg");
list-style-position
Sets the position of the ::marker relative to a list item
list-style-position: inside;
list-style-type
Sets the marker (such as a disc, character, or ustom counter style) of alist item element.
list-style-type: circle;

Pseudo-Classes

A pseudo-class is a selector that selects elements that are in a specific state, e.g., they are the first element of their type, or they are being hovered over by the mouse pointer. They tend to act as if you had applied a class to some part of your document, often helping you cut down on excess classes in your markup, and giving you more flexible, maintainable code.

                        
                            article p:first-child {
                                font-size: 120%;
                                font-weight: bold;
                              }

All pseudo-classes target some bit of your document that is in a certain state, behaving as if you had added a class into your HTML.

Pseudo-Elements

They act as if you had added a whole new HTML element into the markup, rather than applying a class to existing elements. Pseudo-elements start with a double colon ::, such as ::before.

                        
                            article p::first-line {
                                font-size: 120%;
                                font-weight: bold;
                              }
                        
                    

                        
                            <p class="box">Content in the box in my HTML page.</p>

                            .box::after {
                                content: " ➥";
                              }

The ::market pseudo element selects the market box of a list item, which contains the bullet or number. 

                        
                            li::marker {
                                content: "✝ ";
                                font-size: 1.2em;
                              }
