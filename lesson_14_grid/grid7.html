Aligning items in CSS grid layout

CSS grid layout implements CSS box alignment, which is the same standard flexbox uses.

The two axes of a grid layout

When working with grid layout you have two axes available to align things against - the block axis and the inline axis.

The block axis is the axis upon which blocks are laid out in block layout, meaning one below the other. If you have two paragraphs on your page they display one below the other, so it is this direction described as the block axis.

The inline axis runs across the block axis, it is the direction in which text in regular inline flow runs.

Aligning items on the block axis

  • align-self
  • align-items

These properties control alignment on the block axis. When we use these properties, we are changing the alignment of the item within the grid area you have placed it.

Using align-items

Controls alignment on the block axis. The values include

  • normal
  • stretch
  • start
  • end
  • center
  • baseline

Using align-self

This property is set up in a grid cell or grid item, not in the grid container as would be the case for align-items. The values are the same as values of the align-items property.

Items with an intrinsic aspect ratio

The default behavior for align-self is to inherit from the grid container's align-items property, for which the normal default is to stretch, except for items which have an intrinsic aspect ration, in this case they behave as start. The reason for this, is that if items with an aspect ratio are made to stretch, they would be distorted

Justifying items on the inline axis

  • justify-items
  • justify-self

These properties align items on the inline axis. When we use these properties we are changing the alignment of the item within the grid area you have placed it. The values include

  • normal
  • start
  • end
  • left
  • right
  • center
  • stretch
  • baseline

Center an item in the area

Aligning the grid tracks on the block axis

If you have a situation where your grid tracks use an area that is smaller than the grid container you can align the grid tracks themselves inside the container. align-content aligns the grid tracks inside the grid container on the block axis

The align-content property is applied to the grid container as it works on the entire grid.

The values for align-content include:

  • normal
  • start
  • end
  • center
  • stretch
  • space-around
  • space-between
  • space-evenly

Justfying the grid tracks on the inline axis

justify-content aligns the grid tracks inside the grid container on the inline axis.
