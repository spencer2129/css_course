Grid layout using line-based placement

Positioning items by line number

If we do not place items on the grid in any way, they will lay out according to the auto-placement rules, one item in each cell. We can use line-based placement to control where items sit on the grid.


                    .box1 {
                        grid-column-start: 1;
                        grid-column-end: 2;
                        grid-row-start: 1;
                        grid-row-end: 4;
                      }

As you position some items, other items on the grid will continue to be laid out using auto placement rules.

The grid-column and grid-row shorthands

The grid-column-start and grid-column-end properties can be combined into grid-column

The grid-row-start and grid-row-end can be combined into grid-row


                    .box1 {
                        grid-column: 1 / 2;
                        grid-row: 1 / 4;
                      }
                      .box2 {
                        grid-column: 3 / 4;
                        grid-row: 1 / 3;
                      }
                      .box3 {
                        grid-column: 2 / 3;
                        grid-row: 1 / 2;
                      }
                      .box4 {
                        grid-column: 2 / 4;
                        grid-row: 3 / 4;
                      }

Default spans

If an item only spans on track, you can omit the grid-column-end or grid-row-end values.


                    .box1 {
                        grid-column-start: 1;
                        grid-row-start: 1;
                        grid-row-end: 4;
                      }

You can use default spans with grid-column and grid-row 


                    .box1 {
                        grid-column: 1;
                        grid-row: 1 / 4;
                      }

The grid-area property

Instead of using grid-column and grid-row, we can define each area with a grid-area


                    .box1 {
                        grid-area: 1 / 1 / 4 / 2;
                      }
