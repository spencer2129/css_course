The columns CSS property sets the number of columns to use, and the column's width

columns: <column-count> <column-wdith>

Column Properties

column-count
Breaks the element conent into the specified number of columns
column-count: 3
column-width
Sets the dieal column width in a multi-column layout. The container will have as many columns as can fit without any of them having a width less than the column-width value. If the width of the container is narrower than the specified value, the single column's width will be smaller than the declared column width.
column-width: 250px
column-gap
Sets the size of the gap between an element's columns.
column-gap: 2rem
column-gap: 20px
column-gap: 10%
column-span
This property makes it possible for an element to span across all columns when its value is set to all.
column-span: all
column-rule
Sets the width, style, and color of the line drawn between columns in a multi-column layout
column-rule: <column-rule-color> <column-rule-style> <column-rule-width>
column-rule: 5px solid red
column-fill
Controls how an element's content are balanced when broken into columns
column-fill: auto - columns are filled sequentially, and content takes up only the room it needs, possibly resulting in some columns remaining empty
column-fill: balance - default value. Content is equally divided between columns.
