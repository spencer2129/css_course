CSS rules contain declarations, which in turn are composed of properties and values. Each property used in CSS has a value type that describes what kind of values it is allowed to have.

Learn more about CSS values and units

Numbers, lengths, and percentages

<integer>
A whole number such as 1024 or -55
<number>
A decimal number; it may or may not have a decimal pint with a fractional component. For example, 0.255, 128, -1.2
<dimension>
A number with a unit attached to it that includes length, angle, time and resolution types. For example, 45deg, 5s, 10px.
<percentage>
A fraction of some other value, such as 50%. These are always relative to another quantity, such as an element's length being relative to its parent element's length

Length Units

px
Pixels
in
Inches
cm
Centimeters
em
It is a relative lengh unit which is relative to the font size of this element, or the font size of the parent element.
em unit means "my parent element's font-size" if used for font-size and "my own font-size" when used for anything else
rem
It is a relative length unit to the font size of the root element.rem stands for "root em"
vh
It is relative to the viewport's height
vw
It is relative to the viewport's width
