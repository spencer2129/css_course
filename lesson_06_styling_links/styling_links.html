Link States

Links can be in different states and therefore you can use CSS to style those states

Link
A link that has a destination. It can be styled using the :style pesudo class
Visited
A link that has already been visited. It can be styled using the :visited pseudo class
Hover
A link that is hovered over by a user's mouse pointer. It can be styled using the :hover pseudo class
Focus
A link that is focused (moved to by using the tab key). It can be styled using the :focus pseudo class
Active
A link that is activted (clicked on). It can be styled using the :active pseudo class

Default styles

Default links share these properties

  • Links are underlined
  • Unvisited link are blue
  • Visited links are purple
  • Hovering a link makes the mouse pointer change to a small hand icon
  • Focused links have an outline around them.
  • Active links are red

Styling Link

The order of the styling for the diffent link states are very important because link styles build on one another. The styles in the first rule will apply to all s ubsequenet ones.

                        body {
                            width: 300px;
                            margin: 0 auto;

                          }
                          
                          p {

                          }
                          
                          a {

                          a:link {
                            color: #6900ff;
                          }
                          
                          a:visited {
                            color: #a5c300;
                          }
                          
                          a:focus {
                            text-decoration: none;
                            background: #bae498;
                          }
                          
                          a:hover {
                            text-decoration: none;
                            background: #cdfeaa;
                          }
                          
                          a:active {
                            background: #6900ff;
                            color: #cdfeaa;
                          }
